This program demonstrates a help system for Xamarin Forms applications.

The help text uses CSS and you can use a simple HTML editor (I use the free Microsoft Expression Web application) to edit it.

Basically you just add your HTML pages to a "Help" folder in you application and that's it.

Each page (or even each control on a page) can have its own help text.
